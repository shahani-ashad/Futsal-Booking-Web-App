Kandy Sports is a frontend web application designed to facilitate the booking of indoor sports courts (Futsal and Cricket) across the Kandy district. 

The platform allows users to explore different court locations, view pricing, calculate booking costs, and contact management. It features a responsive design and interactive JavaScript validation for a smooth user experience.

Features

* Multi-Location Showcase:Detailed pages for courts in Akurana, Gampola, Gelioya, and Katugastota.
* Interactive Booking System:Select time slots and courts.
    * Automated Price Calculation:JavaScript logic automatically extracts pricing from options and calculates totals.
    * Booking confirmation summaries.
* User Authentication (Frontend):Login and Registration forms.
    * Input validation for emails and passwords.
* Contact Mechanism:Inquiry forms with validation (ensures name, email, and mobile numbers are correct).
    * Direct links to location maps.
* Responsive Design:Optimized for desktop and mobile viewing.

Tech Stack

* Frontend Structure:HTML5
* Styling:CSS3 (Custom `style.css`)
* Interactivity:Vanilla JavaScript (`main.js`)
* Assets:Custom imagery for locations and icons.
